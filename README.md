# Ruchit Chudasama — Portfolio

Personal portfolio site. Built with Vite (vanilla JS, no framework), designed around a "live in the terminal" theme since that's genuinely how I build — git, curl, npm, all day.

## Tech stack
- Vite (vanilla JS)
- Plain CSS with custom properties, no framework
- Fonts: Space Grotesk (display), Inter (body), JetBrains Mono (code/data)

## Running locally
```
npm install
npm run dev
```

## Building for production
```
npm run build
```
Output goes to `dist/`.

## Deployment
Deployed on Netlify. Build command: `npm run build`, publish directory: `dist`.
